# LUXE — Premium E-Commerce UI

A fully-featured, premium Glassmorphism e-commerce interface built with pure HTML5, CSS3, and Vanilla JavaScript.

## Features

- 🎨 Glassmorphism UI with custom purple/pink gradient palette
- 🌙 Dark/Light mode with localStorage persistence
- 🛒 Full cart with quantity controls, tax, shipping, and checkout
- ❤️ Wishlist with sidebar
- 🔍 Live search (name + category)
- 💰 Dual price range slider filter
- 📂 Instant category filtering
- ⚡ Product Quick View modal
- 📦 24 products across 5 categories
- 🎴 Product badges (New, Sale, Hot)
- 💀 Skeleton loading animation
- 📜 Scroll progress bar
- 🔢 Animated counters
- 🍞 Toast notifications
- 📧 Newsletter popup (shown once via localStorage)
- ✅ Fake checkout success popup with SVG animation
- 🔝 Back to Top button
- 📱 Fully responsive (mobile → desktop)
- ♿ Semantic HTML + ARIA attributes
- 🎠 Floating hero shapes + floating card animation

## Tech Stack

- HTML5 (Semantic)
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
